# Netflix Analytics Dashboard

## Overview
Analyzed Netflix Movies & TV Shows dataset using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- WordCloud

## Features
- Data Cleaning
- EDA
- Interactive Dashboard
- Genre Analysis
- Ratings Analysis

## Results
- Movies dominate the catalog.
- USA is the top content producer.
- Drama is the most common genre.
